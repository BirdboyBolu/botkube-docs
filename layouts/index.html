The Kubernetes Bot

Kubernetes bot watches your cluster and notifies of events that you should pay attention to. It also allows you to fire simple commands and get information about resources in cluster.

Configurable

BotKube is configurable so you can watch only objects that you care for with level of severity that you want for events.

Built for Slack

You don't have to leave slack to interact with the Kubernetes cluster

Open Source

BotKube is open source and we welcome your requirements and contributions.
